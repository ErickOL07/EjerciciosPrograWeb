JavaScript Básico

Historia de JavaScript

Cuando empezaron las primeras páginas web en los años 90, solo podían mostrar texto e imágenes. Todo era muy estático, no pasaba nada cuando hacías clic o movías el mouse. Era como ver una hoja de papel en una pantalla.

Entonces, en 1995, en una empresa llamada Netscape, un programador llamado Brendan Eich creó JavaScript.
Su objetivo era hacer que las páginas reaccionaran: que pudieras hacer clic, llenar formularios, mover cosas, mostrar mensajes... ¡hacerlas vivas!

Lo curioso es que lo hizo en solo 10 días.
Así que al inicio, JavaScript era un lenguaje muy básico y con muchos errores, pero sirvió muchísimo para hacer las webs más interesantes.

¿Por qué se llama "JavaScript"?
Porque en ese tiempo estaba de moda otro lenguaje llamado Java, y Netscape pensó que si lo nombraban parecido, se haría popular más rápido.
(Spoiler: Java y JavaScript no tienen casi nada que ver).

Con los años, JavaScript maduró:

  • Se corrigieron errores.
  • Se hicieron reglas más estrictas (como en ECMAScript).
  • Salieron librerías famosas como jQuery que ayudaban a programar más fácil.
  • Después, en 2009, nació Node.js, que permitió usar JavaScript también fuera del navegador, en servidores.
  • Hoy JavaScript se usa para TODO: páginas web, servidores, apps móviles, videojuegos, inteligencia artificial... es uno de los lenguajes más importantes del mundo.

En resumen:

  • Antes las páginas eran planas.
  • JavaScript nació en 1995 para hacerlas interactivas.
  • Al inicio era simple y caótico, pero mejoró mucho.
  • Ahora domina Internet (y más allá).
