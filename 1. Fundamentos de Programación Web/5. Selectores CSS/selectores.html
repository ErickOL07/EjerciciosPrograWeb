CSS Básico

Historia del CSS

CSS, que significa Cascading Style Sheets o "Hojas de Estilo en Cascada", nació porque al principio, cuando se creaban páginas web (en los años 90), todo el diseño —colores, tamaños de letra, márgenes— se hacía directamente en el mismo archivo HTML. Eso hacía que el código fuera un desorden: mezclado todo el contenido (lo que se ve) con la presentación (cómo se ve).

Imagínate querer cambiar el color de todos los títulos de una página grande: tendrías que modificar uno por uno. Un dolor de cabeza.

Entonces, en 1996, un señor llamado Håkon Wium Lie, junto con otros, propuso CSS. Su idea era: separemos el contenido (HTML) de la apariencia (CSS). Así podrías diseñar bonito una vez y aplicarlo a toda la web sin tener que repetir mil veces lo mismo.

En resumen:

  • Antes, el HTML era un caos porque tenía contenido y estilos mezclados.
  • CSS llegó para ordenar, separar y facilitar el diseño.
  • Gracias a CSS, las páginas web ahora pueden ser bonitas, adaptables y mucho más fáciles de mantener.
